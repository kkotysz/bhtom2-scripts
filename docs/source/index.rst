Welcome to bhtom2-scripts documentation!
===================================

**bhtom2-scripts** is a collection of scripts for the BHTOM2 project that uses its API to do multiple tasks.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
