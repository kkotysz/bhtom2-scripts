Usage
=====

.. _installation:

.. Installation
.. ------------

.. To use bhtom2-scripts, first install it using pip:

.. .. code-block:: console

..    (.venv) $ pip install bhtom2-scripts

Executing tasks
----------------
The tasks can be performed by using the script ``bhtom2-scripts.py`` or by importing as a module class named :py:func:`BHTasks`.
